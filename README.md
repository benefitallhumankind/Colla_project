# Colla Project
### 콜라 프로젝트 Created by 질수없조 (이태권, 김미경, 김수빈, 박혜선)

## 기능 설명
* Main
    * 메인 화면
* Member
    * 회원 정보
* WorkSpace
    * 워크스페이스
* Project
    * 프로젝트
* Todo
    * 각 프로젝트마다 To Do 리스트
* ChatRoom
    * 채팅방
* Calendar
    * 일정 관리
* Board
    * 게시판
* Reply
    * 댓글
* Mypage
    * 개인 정보
* Payment
    * 결제
* Notify
    * 알림 기능
